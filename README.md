# Interactive Boxes – Frontend Task

## Description
This project is a Frontend assignment created to demonstrate my understanding of HTML, CSS, and Vanilla JavaScript.
The task focuses on building interactive UI components while following clean code practices and proper structuring.

The application contains three interactive boxes placed vertically (top, middle, and bottom). Each box expands on click and reveals selectable options such as colors and sizes, inspired by the provided example video and Figma design.

## Task Objective
- Showcase basic Frontend development skills
- Implement interactive UI behavior using pure JavaScript
- Replicate the given design and interaction as closely as possible
- Write clean, readable, and well-structured code
- Avoid using any external libraries or frameworks

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## Features
- Three expandable interactive boxes
- Smooth expand and collapse behavior
- Displays color and size options on expansion
- Only one box can be active at a time
- Clean and responsive layout

## Reference
- Example Video: Interactive Boxes Example
- Figma Design: Interactive Boxes Figma Design

The interaction behavior was understood from the example video, and the layout was implemented based on the Figma design.
